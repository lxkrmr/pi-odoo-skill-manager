# osmo v0.3.0

Date: 2026-03-11

## Highlights

- Added explicit `--project <path>` targeting for runtime commands.
- Commands now supporting `--project`:
  - automation: `components`, `enable-skill`, `disable-skill`
  - human-ops: `up`, `db`, `shell`, `test`, `lint`
- Added shared resolver to keep path handling deterministic and validated.
- Extended CLI contract tests to cover runtime `--project` behavior.
- Updated docs and delivery plan for v0.3.0 scope.
